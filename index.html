Meteor Italia

Perché scrivere il codice javascript per client e server, quando puoi scriverlo una volta sola?

Cos'é "Meteor Italia" ?

Meteor Italia è un gruppo informale per promuovere l'utilizzo della piattaforma open source Meteor, degli sviluppatori e delle aziende/startup/iniziative che lo utilizzano.

Cos'é Meteor o MeteorJS ?

  1. Isomorfico: il codice javascript scritto può funzionare in modo trasparente sul client (browser), sul server (NodeJS) o in entrambi i mondi.
  2. Latency compensation: o anche "compensazione di lantenza". Significa che le azioni vengono eseguite immediatamente lato client, ma il server può eventualmente fare rollback se è stata fatta un'azione non valida.
  3. Reattività e Virtual DOM: Le viste vengono automaticamente ricalcolate al variare dei dati e solo le parti di HTML che cambiano vengono effettivamente sostituite.
  4. Molte tecnologie che già conosci: MeteorJS include molte altre librerie e tecnologie open source già conosciute: jquery, underscore, NodeJS, MongoDB, MiniMongo,..
  5. Ecosistema e modularità: la comunità di Meteor è molto attiva e molte funzionalità client o server potrebbero già essere pacchettizzate dal package manager ufficiale. Fatevi un giro su Atmosphere

Maggiori informazioni su Meteor si possono trovare sul sito ufficiale Meteor (in inglese), oppure nella documentazione tradotta dai volontari italiani:

  1. Meteor Docs. La documentazione ufficiale tradotta in italiano
  2. Discover Meteor. La versione in italiano del libro su Meteor più seguito. Disponibile in HTML e scaricabile gratis in Ebook PDF.
